## 二、新增功能

### 2.1 首页 - 今日推荐菜谱
- 首页显示今日推荐的菜谱
- 支持换一批刷新推荐
- 点击可查看详情

### 2.2 食材模块

#### 2.2.1 图片上传功能
- 支持拍照上传
- 支持从相册选择
- 图片压缩后上传
- 最大支持1张图片

#### 2.2.2 扫描录入食材
- 调用微信扫码API
- 支持条码/二维码识别
- 预留OCR接口（需配置阿里云OCR）

#### 2.2.3 列表信息扩展
- 显示食材图片
- 显示创建人
- 显示创建时间
- 显示过期时间
- 状态标签（新鲜/临期/已过期）

#### 2.2.4 临期/过期食材筛选
- 新增状态筛选：全部/新鲜/临期/已过期
- 临期定义：3天内过期
- 已过期定义：已过保质期

### 2.3 食谱模块

#### 2.3.1 图片上传功能
- 支持多张图片上传（最多5张）
- 支持拍照和相册选择
- 图片轮播展示

#### 2.3.2 烹饪扣减库存
- 食谱详情页增加"开始烹饪"按钮
- 显示所需食材清单
- 自动扣减对应食材库存
- 库存不足时提示

### 2.4 购买食材模块（原购物清单）

#### 2.4.1 模块更名
- "购物清单" → "购买食材"

#### 2.4.2 分类字段
- 支持分类：蔬菜/肉类/海鲜/蛋奶/水果/调料/主食/其他

#### 2.4.3 已购买/未购买状态
- 复选框标记购买状态
- 已购买项显示删除线
- 支持一键全选/取消

#### 2.4.4 一键入库功能
- 选择已购买食材
- 设置存放位置（冷藏/冷冻/常温）
- 设置过期时间
- 批量添加到食材库

#### 2.4.5 历史购物清单
- 新增"历史"Tab
- 按日期分组显示
- 显示购买日期、商品数量

#### 2.4.6 再次购买
- 从历史清单快速添加商品
- 一键添加到当前购买清单

## 三、后端开发

### 3.1 新增接口

| 接口 | 方法 | 路径 | 说明 |
|-----|------|------|------|
| 图片上传 | POST | /api/v1/upload/image | 上传图片到OSS |
| OCR识别 | POST | /api/v1/ocr/recognize | 识别图片文字 |
| 食材筛选 | GET | /api/v1/foods/filter | 按状态筛选食材 |
| 批量入库 | POST | /api/v1/shopping/batch-stock | 一键入库 |
| 烹饪扣减 | POST | /api/v1/recipes/{id}/cook | 选择食谱扣减库存 |
| 购物历史 | GET | /api/v1/shopping/history | 获取历史清单 |
| 再次购买 | POST | /api/v1/shopping/repurchase/{id} | 从历史清单添加 |

### 3.2 数据库变更

#### 3.2.1 foods 表 - 新增字段
```sql
ALTER TABLE foods ADD COLUMN image_url VARCHAR(500);
ALTER TABLE foods ADD COLUMN creator_id BIGINT;
ALTER TABLE foods ADD COLUMN creator_name VARCHAR(50);
```

#### 3.2.2 recipes 表 - 新增字段
```sql
ALTER TABLE recipes ADD COLUMN images JSON;
```

#### 3.2.3 shopping_items 表 - 重构
```sql
ALTER TABLE shopping_items ADD COLUMN category VARCHAR(20);
ALTER TABLE shopping_items ADD COLUMN status TINYINT DEFAULT 0;
ALTER TABLE shopping_items ADD COLUMN storage_location VARCHAR(20);
ALTER TABLE shopping_items ADD COLUMN expire_date DATE;
ALTER TABLE shopping_items ADD COLUMN purchased_at DATETIME;
ALTER TABLE shopping_items ADD COLUMN archived_at DATETIME;
```

#### 3.2.4 新增 shopping_history 表
```sql
CREATE TABLE shopping_history (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    total_items INT DEFAULT 0,
    total_amount DECIMAL(10,2),
    items JSON,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

## 六、配置说明

### 6.1 后端配置

编辑 `application-prod.yml`:

```yaml
# 阿里云OSS配置（用于图片上传）
aliyun:
  oss:
    endpoint: oss-cn-hangzhou.aliyuncs.com
    accessKeyId: your-access-key-id
    accessKeySecret: your-access-key-secret
    bucketName: your-bucket-name

# 阿里云OCR配置（用于扫描录入）
aliyun:
  ocr:
    accessKeyId: your-access-key-id
    accessKeySecret: your-access-key-secret
```

### 6.2 微信小程序配置

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入"开发" → "开发管理" → "开发设置"
3. 配置服务器域名：
   - request合法域名: `https://api.hzqstudio.store`
   - uploadFile合法域名: `https://api.hzqstudio.store`
