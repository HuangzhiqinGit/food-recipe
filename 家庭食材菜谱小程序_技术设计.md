## 6. 后端技术方案

### 6.1 OSS图片上传

```java
@Service
public class OssService {
    
    @Value("${aliyun.oss.endpoint}")
    private String endpoint;
    
    @Value("${aliyun.oss.accessKeyId}")
    private String accessKeyId;
    
    @Value("${aliyun.oss.accessKeySecret}")
    private String accessKeySecret;
    
    @Value("${aliyun.oss.bucketName}")
    private String bucketName;
    
    public String upload(MultipartFile file, String type) {
        // 生成文件名
        String fileName = type + "/" + UUID.randomUUID() + ".jpg";
        
        // 上传
        OSS ossClient = new OSSClientBuilder().build(endpoint, accessKeyId, accessKeySecret);
        try {
            ossClient.putObject(bucketName, fileName, file.getInputStream());
            return "https://" + bucketName + "." + endpoint + "/" + fileName;
        } finally {
            ossClient.shutdown();
        }
    }
}
```

### 6.3 库存扣减（乐观锁）

```java
@Service
public class FoodService {
    
    @Transactional
    public void deductStock(Long foodId, Integer quantity) {
        // 使用乐观锁防止并发问题
        int rows = foodMapper.deductStock(foodId, quantity);
        if (rows == 0) {
            throw new BusinessException("库存不足");
        }
    }
}

// Mapper
@Update("UPDATE foods SET quantity = quantity - #{quantity}, version = version + 1 " +
        "WHERE id = #{foodId} AND quantity >= #{quantity}")
```
