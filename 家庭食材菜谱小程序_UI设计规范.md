# 家庭食材&菜谱管理小程序 - UI设计规范 v1.0

> **文档版本**: v1.0  
> **设计工具**: 基于微信小程序设计规范  
> **适用项目**: 家庭食材&菜谱管理小程序

---

## 1. 设计概述

### 1.1 设计理念
- **清新自然**: 以绿色为主色调，传达新鲜、健康的食材理念
- **简洁高效**: 信息层级清晰，操作路径最短化
- **温暖亲切**: 圆角设计，柔和配色，营造家庭温馨感

### 1.2 设计风格
| 维度 | 定位 |
|------|------|
| 整体风格 | 清新、现代、扁平化 |
| 视觉密度 | 中等密度，适当留白 |
| 圆角风格 | 统一12rpx圆角 |
| 阴影运用 | 轻量投影，营造层次感 |

## 6. 页面设计详解

### 6.1 首页

**布局结构**:
```
┌─────────────────────────────────────┐
│ 状态栏 (44px)                       │
├─────────────────────────────────────┤
│  你好，张三！                       │  ← 欢迎语 (32rpx, #212121)
│  今天想吃点啥？                     │  ← 副标题 (28rpx, #616161)
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐   │
│  │  ⚠️ 临期食材提醒             │   │  ← 提醒卡片
│  │  有3个食材即将过期           │   │
│  │  [点击查看详情 →]            │   │
│  └─────────────────────────────┘   │
│  (背景: rgba(255, 152, 0, 0.1))     │
├─────────────────────────────────────┤
│  今日菜谱推荐              [换一批] │  ← 模块标题
├─────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐        │
│  │ [图片]   │  │ [图片]   │        │  ← 菜谱卡片网格
│  │ 番茄炒蛋 │  │ 青椒肉丝 │        │
│  │ [全素]   │  │ [荤]     │        │
│  └──────────┘  └──────────┘        │
├─────────────────────────────────────┤
│  [首页] [食材] [菜谱] [清单]        │  ← TabBar
└─────────────────────────────────────┘
```

**设计要点**:
- 欢迎语使用大号字体，亲切友好
- 临期提醒使用橙色背景卡片，醒目但不突兀
- 菜谱卡片使用2列网格布局
- "换一批"使用文字按钮样式

```

│  │ ● 鸡蛋                 >    │   │  ← 食材卡片
│  │ 5个 | 冰箱冷藏              │   │
│  │ [新鲜]                     │   │
│  └─────────────────────────────┘   │

│  └─────────────────────────────┘   │

│  [首页] [食材] [菜谱] [清单]        │
```

**设计要点**:
- 搜索框圆角设计，左侧搜索图标
- 新增按钮使用橙色圆形按钮，带+图标
- 分类标签横向滚动，选中状态使用主色
- 食材卡片使用白色背景+轻微阴影
- 状态指示点：新鲜(绿)/临期(橙)/过期(红)

```
│  ┌──────────────┬──────────┐      │
│  │ 5            │ 个    ▼  │      │  ← 数字+单位
│  └──────────────┴──────────┘      │
│                                     │
│  存放位置                           │
│  ┌─────────────────────────────┐   │
│  │ 冰箱冷藏             ▼      │   │
│  └─────────────────────────────┘   │
```

**设计要点**:
- 表单标签左对齐，必填项标*
- 输入框使用灰色背景，聚焦时绿色边框
- 常用食材使用标签形式，点击快速填充
- 底部按钮固定，取消(灰底)+保存(绿底)
