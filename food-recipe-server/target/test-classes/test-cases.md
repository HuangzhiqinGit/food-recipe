## 3. 接口测试用例

### 3.1 认证接口

| 接口 | 方法 | 测试场景 | 预期结果 |
|------|------|----------|----------|
| /api/v1/auth/login | POST | 正确code | 返回token和用户信息 |
| /api/v1/auth/login | POST | 错误code | 返回错误信息 |
| /api/v1/auth/validate | GET | 有效token | 返回用户信息 |
| /api/v1/auth/validate | GET | 无效token | 返回401错误 |

### 3.2 食材接口

| 接口 | 方法 | 测试场景 | 预期结果 |
|------|------|----------|----------|
| /api/v1/foods | GET | 获取列表 | 返回食材列表和临期数量 |
| /api/v1/foods | GET | 带分类筛选 | 返回筛选后的列表 |
| /api/v1/foods | GET | 带关键词搜索 | 返回匹配的列表 |
| /api/v1/foods | POST | 新增食材 | 返回新增食材信息 |
| /api/v1/foods/{id} | PUT | 编辑食材 | 返回更新后的食材 |
| /api/v1/foods/{id} | DELETE | 删除食材 | 返回成功 |
| /api/v1/foods/expiring/count | GET | 获取临期数量 | 返回临期食材数量 |

### 3.3 菜谱接口

| 接口 | 方法 | 测试场景 | 预期结果 |
|------|------|----------|----------|
| /api/v1/recipes | GET | 获取列表 | 返回菜谱列表 |
| /api/v1/recipes | GET | 按库存匹配 | 返回匹配度排序的列表 |
| /api/v1/recipes | GET | 带筛选条件 | 返回筛选后的列表 |
| /api/v1/recipes/{id} | GET | 获取详情 | 返回菜谱详情 |
| /api/v1/recipes | POST | 添加菜谱 | 返回新增菜谱 |
| /api/v1/recipes/{id}/favorite | POST | 收藏/取消收藏 | 返回收藏状态 |
| /api/v1/recipes/favorites | GET | 获取收藏列表 | 返回收藏的菜谱 |

### 3.4 购物清单接口

| 接口 | 方法 | 测试场景 | 预期结果 |
|------|------|----------|----------|
| /api/v1/shopping | GET | 获取清单 | 返回清单列表 |
| /api/v1/shopping | POST | 添加清单项 | 返回新增清单项 |
| /api/v1/shopping/{id}/toggle | PUT | 切换购买状态 | 返回成功 |
| /api/v1/shopping/{id} | DELETE | 删除清单项 | 返回成功 |
| /api/v1/shopping/clear | DELETE | 清空清单 | 返回成功 |
